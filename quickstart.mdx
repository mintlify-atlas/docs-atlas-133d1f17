## Prerequisites

Before you begin, ensure you have the following installed:

- **Python 3.8+** with pip
- **Node.js 16+** with npm
- **FFmpeg** (for video processing)
- **Git** (to clone the repository)

You'll also need API keys from:
- [Google AI Studio](https://makersuite.google.com/app/apikey) (for Gemini API)
- [Sarvam AI](https://www.sarvam.ai/) (for text-to-speech)
- [Unsplash](https://unsplash.com/developers) (for images)

## Get Started in 5 Minutes

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/Kamal-Nayan-Kumar/AI-Video-Gen
    cd AI-Video-Gen
    ```
  </Step>

  <Step title="Set up the backend">
    Navigate to the backend directory and create a virtual environment:
    
    ```bash
    cd backend
    python -m venv venv
    ```

    Activate the virtual environment:
    
    <CodeGroup>
      ```bash Windows
      venv\Scripts\activate
      ```

      ```bash macOS/Linux
      source venv/bin/activate
      ```
    </CodeGroup>

    Install dependencies:
    
    ```bash
    pip install -r requirements.txt
    ```

    Install Manim (animation library):
    
    ```bash
    pip install manim
    ```
  </Step>

  <Step title="Configure environment variables">
    Create a `.env` file in the `backend/` directory:
    
    ```bash
    cp .env.example .env
    ```

    Edit `.env` and add your API keys:
    
    ```bash
    GEMINI_API_KEY=your_gemini_api_key_here
    SARVAM_API_KEY=your_sarvam_api_key_here
    UNSPLASH_ACCESS_KEY=your_unsplash_access_key_here
    ```

    <Note>
      The backend uses Sarvam AI for text-to-speech. Make sure to get your API key from [Sarvam AI](https://www.sarvam.ai/).
    </Note>
  </Step>

  <Step title="Start the backend server">
    With your virtual environment still activated, run:
    
    ```bash
    python app.py
    ```

    The FastAPI backend will start on `http://localhost:8000`. You should see:
    
    ```
    INFO:     Started server process
    INFO:     Uvicorn running on http://0.0.0.0:8000
    ```
  </Step>

  <Step title="Set up the frontend">
    Open a new terminal and navigate to the frontend directory:
    
    ```bash
    cd frontend
    ```

    Install dependencies:
    
    ```bash
    npm install
    ```

    Start the development server:
    
    ```bash
    npm run dev
    ```

    The React frontend will start on `http://localhost:5173`.
  </Step>

  <Step title="Generate your first presentation">
    1. Open your browser and go to `http://localhost:5173`
    2. Enter a topic (e.g., "Introduction to Machine Learning")
    3. Set the number of slides (e.g., 5)
    4. Choose a language (English, Hindi, Kannada, or Telugu)
    5. Select a tone (Formal, Casual, or Educational)
    6. Click **Generate**

    The system will:
    - Generate presentation content using Gemini AI
    - Create narration scripts for each slide
    - Generate voice audio using Sarvam AI
    - Fetch relevant images from Unsplash or create animations with Manim
    - Compose everything into a final MP4 video

    <Note>
      The generation process takes 2-5 minutes depending on the number of slides and complexity.
    </Note>
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Backend server won't start">
    Make sure:
    - Your virtual environment is activated
    - All dependencies are installed: `pip install -r requirements.txt`
    - FFmpeg is installed and accessible in your PATH
    - Port 8000 is not already in use
  </Accordion>

  <Accordion title="Frontend can't connect to backend">
    - Verify the backend is running on `http://localhost:8000`
    - Check browser console for CORS errors
    - Ensure your `.env` file has the correct API keys
  </Accordion>

  <Accordion title="Video generation fails">
    Common issues:
    - Invalid API keys in `.env`
    - FFmpeg not installed or not in PATH
    - Insufficient disk space in `backend/outputs/`
    - API rate limits exceeded
  </Accordion>
</AccordionGroup>
